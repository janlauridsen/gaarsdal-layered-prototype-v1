# Decision Case Catalog

Dette katalog indeholder beslutnings- og grænsecases
for gaarsdal-layered-prototype-v1.

Formålet er at:
- gøre AI-adfærd testbar
- fastholde etiske grænser
- opdage arkitektonisk glidning tidligt

Kataloget tester:
- input → signal-klassifikation
- signaler → strategi
- strategi → orkestrering

Kataloget tester ikke:
- UI-tekst
- brugeroplevelse
- hjælpsomhed
- effektivitet

Dette katalog er normativt og versioneret
på linje med systemets øvrige kontrakter.
